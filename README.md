A small learning platform focused on connecting theory with practice. This repository contains a React + TypeScript app built with Vite and Tailwind CSS.

## Local development

Requirements: Node.js (or your runtime of choice) and npm.

```sh
# clone
git clone https://github.com/HuyDinhdmm/manabi-zen-learn.git
cd manabi-zen-learn

# install
npm install

# run dev server
npm run dev
```

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui (UI components)
